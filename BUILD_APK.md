# Как собрать APK через GitHub Actions (без Android Studio)

## Быстрый план
1. Сделать репозиторий на GitHub
2. Залить туда проект
3. GitHub сам соберёт APK (5-10 минут)
4. Скачать APK, поставить на телефон

---

## Шаг 1 — регистрация на GitHub

Идёшь на https://github.com → **Sign up** если аккаунта нет (email + пароль, 2 минуты).

## Шаг 2 — создание репозитория

1. Кнопка **"+"** в правом верхнем углу → **New repository**
2. Имя: `esp32-audio` (любое)
3. **Public** (для бесплатной сборки Actions должен быть публичный, либо Private + тратить бесплатные минуты, их хватает с головой)
4. Галочку "Add a README" **НЕ** ставь
5. **Create repository**

## Шаг 3 — загрузка файлов

Самый простой способ — **через веб-интерфейс GitHub**:

1. На пустом репозитории жми **"uploading an existing file"** (ссылка в тексте)
2. Перетащи **всю папку `wifi_audio`** в окно загрузки
3. Внизу: **Commit changes** → **Commit directly to main** → **Commit changes**

Альтернатива через **GitHub Desktop** (если удобнее GUI):
1. Скачать https://desktop.github.com/
2. Войти в аккаунт
3. File → Clone repository → свой только что созданный esp32-audio
4. Скопировать содержимое папки `wifi_audio` в клонированную папку
5. Commit + Push в интерфейсе GitHub Desktop

## Шаг 4 — запуск сборки

После push:
1. Открой свой репозиторий на GitHub
2. Вкладка **Actions** (сверху)
3. Увидишь запущенную сборку **"Build APK"**
4. Если не запустилась сама — кликай "Build APK" слева → **Run workflow** → **Run workflow**
5. Ждёшь 5-10 минут (зелёная галочка = готово)

## Шаг 5 — скачивание APK

1. Кликни на завершённую сборку
2. Внизу страницы раздел **Artifacts**
3. Скачай архив **ESP32Audio-debug-apk** (zip)
4. Распакуй → внутри файл **app-debug.apk**

## Шаг 6 — установка на телефон

1. Перекинь apk на телефон (USB, Telegram себе, Google Drive)
2. На телефоне открой файл
3. Android скажет "установка из неизвестных источников запрещена" → Настройки → разрешить для файлового менеджера или браузера
4. Установи

## Шаг 7 — запуск

1. Подключи телефон к WiFi **ESP32-Audio** (пароль `12345678`)
2. Запусти приложение
3. IP `192.168.4.1`, порт `3333` (уже вписаны)
4. **Старт** → разреши захват звука
5. Включай YouTube / Spotify — звук пойдёт на XH-M177

---

## Если что-то пошло не так при сборке

**Смотри логи сборки** — на странице Actions кликаешь на упавшую сборку, открываешь шаг "Build debug APK" — там увидишь ошибку.

Типичные причины:
- Забыл залить какой-то файл — проверь что все папки из `wifi_audio/android_app/` загружены
- Файл `gradlew` потерял exec-права — на такой случай в workflow уже стоит `chmod +x ./gradlew`
- Слишком старый JDK в workflow — стоит 17, этого хватит

Скинешь лог ошибки — разберёмся.

## Бесплатные минуты GitHub Actions

- Public-репозиторий: **unlimited**
- Private: 2000 минут/месяц бесплатно (одна сборка ~5 мин → 400 сборок)
